# FormosanBank

FormosanBank is a machine-readable corpus and tooling ecosystem for Taiwan's Indigenous Formosan languages. The project brings together text, audio, metadata, annotations, and developer tools so these endangered languages can be used in linguistic research, education, revitalization work, and language technology.

## What We Build

- Corpus repositories for dictionaries, transcriptions, historical texts, media, Bible materials, educational resources, and academic datasets.
- Standardized XML data with metadata, citations, translations, glosses, dialect labels, and audio alignments where available.
- Hugging Face datasets, models, and demos for ASR, machine translation, NLP, and public data access.
- Documentation for corpus structure, source provenance, licensing, contribution workflows, and developer use.

## Use and Citation

Licensing and citation requirements vary by corpus. Before reusing data, check the repository README, corpus documentation, and FormosanBank terms for the specific resource you are using.

FormosanBank is built with contributors, researchers, and language communities. If you want to contribute data, tools, corrections, or documentation, start with the contribution guide above.
